This case study involved creating a cooking app that encompassed UX and UI methods.

SimplyCook is a responsive web application to help users search and save recipes, and follow clear step-by-step instructions through a user-friendly interface. The app is designed to encourage users to become more comfortable in the kitchen.

Tools Used: Adobe XD

My Process

In order to identify the appropriate designs for this app, I followed a structured process in my design approach below:

  1. Research on existing apps
  2. Create user personas & user flows
  3. Design wireframes for user testing
  4. Implement final designs from style guide

Research

I conducted research through a competitive analysis to identify the purpose, strengths and opportunities of existing apps. Below is the summary of this analysis.

User Personas & User Flow

Based on the research, user personas were created and the user flow was identified to understand the tasks to be done for the SimplyCook app.

Wireframes & User Testing

I built out low-fidelity wireframes for quick prototype testing on the screen flows. Based on the feedback received, I updated my designs in the mid-fidelity wireframes.

Style Guide & Final Designs

A style guide was created to set the expectations of the app design and the imagery to convey when a user is interacting with the app. In addition, I built out the responsive designs for the screens to be displayed on various devices.

Reflections

I enjoyed conducting the research on the existing apps to identify areas of opportunity to include in this app. Once those were determined, creating the quick prototype was really great to see users interacting with the screens and see it come to life from some quick sketches.

Some of the biggest challenges was narrowing down the list of features I wanted to incorporate in the app. I may build out more features that can help the users stay more engaged in using the app.
